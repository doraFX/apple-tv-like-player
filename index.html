Overview

简介

一个轻量级原生 JS 视频播放器插件，可将普通 <video> 元素无缝增强为具有 Apple TV 风格交互美学体验的高级播放器。丝滑流畅，无需繁杂配置。

Quick Start

使用方法

<link rel="stylesheet" href="apple-video-player.css">
<script src="apple-video-player.js"></script>

<video src="demo.mp4"></video>
✨ 加载后会自动增强所有视频元素。
