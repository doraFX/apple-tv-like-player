# 🍎 Apple TV 风格 HTML5 视频播放器

🍎 Apple TV 风格视频播放器组件
一个轻量、流畅、开箱即用的视频播放器 UI。

---

## 📖 简介

这是一个基于原生 JavaScript 的轻量级插件，可以将普通的 `<video>` 元素增强为具有 Apple TV 风格的沉浸式播放器。

无需初始化，加载即用，自动作用于页面中的所有视频元素。

---

## 🌐 在线预览

👉 https://dorafx.github.io/apple-tv-like-player/

---

## ✨ 特性

* Apple TV 风格控制界面
* 自动增强页面中的 video
* 空闲自动隐藏控制层
* 平滑拖动进度条
* 音量图标智能分级
* 全屏过渡动画
* 键盘快捷键支持
* 移动端兼容

---

## 🚀 使用方式

```html
<link rel="stylesheet" href="apple-video-player.css">
<script src="apple-video-player.js"></script>

<video src="demo.mp4"></video>
```

无需额外初始化，加载后自动生效。

---

## ⌨️ 快捷键

| 键位    | 功能      |
| ----- | ------- |
| Space | 播放 / 暂停 |
| Enter | 切换全屏    |
